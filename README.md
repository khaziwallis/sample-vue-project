# sample-vue-project

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```


## build and run over serve
````
yarn build
yarn global add serve
serve -s dist
```
